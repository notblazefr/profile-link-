## Features
* **Vanta.net Background:** Interactive 3D particle net that reacts to mouse movement.
* **Dual Cursor System:** Custom dot cursor combined with a radial spotlight effect.
* **Typewriter Effect:** Smooth auto-typing animation for bio and status messages.
* **MUSIC:** Interactive "Click to Access" overlay that triggers background music.
* **Glassmorphism:** Modern frosted-glass card design using CSS backdrop-filters.

---

## Customization Guide
1. **Open** `index.html`.
2. **Tab Title:** Change line 6 (`<title>...</title>`).
3. **Music:** Update line 112 with your direct MP3 link.
4. **Avatar:** Update line 116 with your profile picture URL.
5. **Socials:** Update lines 124-126 with your Discord, GitHub, and Instagram usernames.
6. **Bio Text:** Modify line 147 inside the script tag to change the typewriter text.
